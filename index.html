Content Area

  • Contains the actual data, such as text, images, or other media.
  • Sized using the width and height properties.
  • Bounded by the content edge.

Padding Area

  • Surrounds the content area.
  • Space within the border box.
  • Dimensions are determined by the width and height of the padding box.

Margin Area

  • Separates the element from neighboring elements.
  • Dimensions specified by the margin-box width and height.

Media Query ruleset

@media only screen and (/*condition goes here */)
{ body {
/* Conditions goes here */
} }

Explanation

The above is the general way to call media queries. In the parenthesis, we add the conditions for the browser windows such as what is the minimum/maximum size you want the browser window to be for the conditions in the “body” to appear

Example

@media only screen and (max-width: 500px) {
body {
background-color: black;
}
}
