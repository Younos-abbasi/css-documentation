The CSS box model is essentially a box that wraps around every HTML element.

Content Area

  • Contains the actual data, such as text, images, or other media.
  • Sized using the width and height properties.
  • Bounded by the content edge.

Padding Area

  • Surrounds the content area.
  • Space within the border box.
  • Dimensions are determined by the width and height of the padding box.

Margin Area

  • Separates the element from neighboring elements.
  • Dimensions specified by the margin-box width and height.

The @media rule, introduced in CSS2, made it possible to define different style rules for different media types.

Media Query ruleset

@media only screen and (/*condition goes here */)
{ body {
/* Conditions goes here */
} }

Explanation

The above is the general way to call media queries. In the parenthesis, we add the conditions for the browser windows such as what is the minimum/maximum size you want the browser window to be for the conditions in the “body” to appear

Example

@media only screen and (max-width: 500px) {
body {
background-color: black;
}
}

A CSS pseudo-element is used to style specified parts of an element

Pseudo Elements syntax

selector::pseudo-element {
property: value;
}

Explanation

A CSS pseudo-element is a keyword added to a selector that lets you style a specific part of the selected elements.

Examples

/* unvisited link */
a:link {
color: #FF0000;
}

CSS Transform

Explanation

The transform property allows you to visually manipulate an element by skewing, rotating, translating, or scaling:

Example

.element {
width: 20px;
height: 20px;
transform: scale(20) skew
(-20deg);
}

Values

  • Scale
  • Skew
  • Translate
  • Rotate
  • Matrix
  • Prospective

CSS animations make it possible to animate

CSS ANimations

An animation lets an element gradually change from one style to another. You can change as many CSS properties you want, as many times as you want. To use CSS animation, you must first specify some keyframes for the animation. Keyframes hold what styles the element will have at certain times.

@keyframes Rule

When you specify CSS styles inside the @keyframes rule, the animation will gradually change from the current style to the new style at certain times.

properties

  • @keyframes
  • animation-name
  • animation-duration
  • animation-delay
  • animation-iteration-count
  • animation-direction
  • animation-timing-function
  • animation-fill-mode
